Oh-My-Zsh plugins
=================

* cd ~/.oh-my-zsh/custom/
* rm -rf plugins
* git clone https://github.com/sasik-github/plugins.git
* mcedit ~/.zshrc
* add to plugins line
```
    plugins=(sasik-apt)
```

sasik-apt plugin
----------------

shortcut for Debian apt

* sagi - sudo apt-get install
* sagu - sudo apt-get update
* sagr - sudo apt-get remove
* sagug - sudo apt-get upgrade
* sacs - sudo apt-cache search
